# クローン声でポッドキャスト作成ツール
利用者の声をクローンして入力した台本（テキスト）でポッドキャストを作成する。

## 利用者
自分のみ（他人は使わないので認証は不要）

# 機能

## 声の管理
声の登録 … TTSにクローンさせる声を録音して保存する
声の削除 … 登録した声を削除する
声の一覧 … 登録済の声を一覧にする
声の再生 … 登録済の声を試聴する

## ポッドキャスト作成
台本入力 … ポッドキャストの台本を入力する
声の選択 … ポッドキャストの声を一覧から選択する
BGM合成 … ポッドキャストのBGMを入力する
ポッドキャスト作成 … 台本のTTSとBGM合成を行いダウンロード可能なポッドキャスト音声(mp3)を作成する。

## 利用技術
- Vue.js … ローカルWebアプリ
- qwen3-tts … TTS。ReplicateにAPIがある（https://replicate.com/qwen/qwen3-tts）
- ffmpeg … BGM合成

# 品質保証
- 単体テストは行う
- E2EテストはPlaywrightで主要シナリオについて行う。
